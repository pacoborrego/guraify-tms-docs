2.3 Traceability Model
======================

Traceability in Guraify TMS is structured across three layers:

1. Goods traceability (packages)
2. Event traceability (stops)
3. Document traceability (POD, incidents)

Each entity maintains its full historical record.

The system allows real-time visibility of:

- Where the goods are
- Their operational state
- Which resource transports them
- Any incidents occurred
