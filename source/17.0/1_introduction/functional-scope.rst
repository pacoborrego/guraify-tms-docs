1.2 System Functional Scope
===========================

In Scope
--------

- Shipment ingestion (manual / file import / API-EDI)
- Data model transformation into execution units (legs, stops, trips)
- Planning and assignment workflows
- Mobile execution (POD, incidents, scanning)
- Traceability events and operational state transitions
- Carrier settlement and invoicing support

Out of Scope
------------

- Third-party systems internal behavior (only the interface contract is documented)
- Customer-specific developments not part of the baseline product
- Generic Odoo accounting beyond what is required by TMS processes
