1.1 Document Purpose
====================

Overview
--------

This document is the technical and functional reference for **Guraify TMS** (Odoo 17.0).

Objectives
----------

- Provide a shared understanding of the TMS conceptual model and operational workflow.
- Describe end-to-end processes (planning → execution → closing → settlement).
- Define configuration guidelines for consistent deployments.
- Document integration strategy (EDI/API/Webhooks) and extension points.
- Serve as a baseline for onboarding, support, and future upgrades.
