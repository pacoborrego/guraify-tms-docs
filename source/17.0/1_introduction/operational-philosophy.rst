1.5 Operational Philosophy
==========================

Core Idea
---------

The TMS is designed to eliminate operational uncertainty by making execution state explicit, traceable,
and consistent across planning, mobile execution, and administrative workflows.

Principles
----------

- Model-first: orders are transformed into execution units (legs/stops/trips).
- Controlled state transitions across operational entities.
- Traceability is first-class: every relevant event is recorded with timestamp and context.
- Separation of concerns: planning vs execution vs administration.
- Configuration over code where possible.
